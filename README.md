
# Build Jar 
***
1. Clone the Repository 
2. Go to Repository
3. Build using following command
```
./gradlew build
```
**Note :- Already build jar is in Jar folder**
1. Jar will be created in build/libs folder with name **quantcastKotlin-1.0-SNAPSHOT-all.jar**

2. run using following command replace filepath and date in command
```
java -jar quantcastKotlin-1.0-SNAPSHOT-all.jar -f <filepath> -d <date>
```
